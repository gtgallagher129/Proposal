Proposal “Landmines”

I am choosing to do a project in javascript. It will be very much like the game minesweeper.

Application Overview

This will be a game. There will be a certain number of mines scattered amount a grid. The player's goal will be to dig out all non-mine squares, and find the position of all mines. Once that is done, the game is won.

Technology Overview

This game is going to be coded through javascript and played within any terminal window. I will likely have to import a random number generator. Because it will be on javascript I will host through Azure.
